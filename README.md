# Personal Portfolio Website

## Overview
This project is a personal portfolio website created to showcase my background, technical skills, and projects. The site serves as a professional landing page that highlights my experience, resume, and academic work.

The portfolio was built using modern HTML and CSS principles with a focus on clean layout and responsiveness.

## Usage Instructions
Upon visiting the website, users are greeted by a welcome page. Navigation links allow users to explore:
- Projects
- Resume
- Personal/About section

Each page provides structured information about my work and experience.

## Installation Instructions
No installation is required. This portfolio is deployed via GitHub Pages and can be accessed directly at:

**[https://kolivas380.github.io/]**

## Technologies Used
- HTML5
- CSS3
